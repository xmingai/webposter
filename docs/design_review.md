## 🔴 严重问题

### 1. 左侧栏工具按钮 — 图标和文字被截断/偏移
- 图标（MousePointer2、Upload 等）被推到了 **侧边栏最左边缘**，几乎贴到边框
- 这是因为 `Button` 组件的 `justify-start` 加上 `w-full` 但 **侧边栏没有设置内边距**，导致图标溢出到边界
- 对比理想状态：图标应该有 ~14px 左侧内边距

### 2. "快速素材" 标题 — 完全无左边距
- `快速素材` 标题文字和图片缩略图网格都紧贴侧边栏左边缘
- 没有呼吸空间，视觉上非常压迫

### 3. 右侧 AI 面板 — 缺少边界分隔
- AI 助手和输入框 **没有明确的视觉分隔线** 与画布区域之间
- 实际上有 `border-l border-border`，但在暗色模式下 border 几乎不可见（oklch 10% opacity 太低）

### 4. 底部工具栏 — 无背景色区分
- 底部工具栏的撤销/重做/删除图标直接浮在最底部
- 没有和画布区域形成明显的区隔，看起来像浮动图标而非工具栏
- "导出" 按钮虽然有颜色，但和同排其他元素风格断裂

## 🟡 视觉不一致

### 5. 重复的缩放显示
- 画布右下角有一个 `100%` 浮动 badge
- 底部工具栏右侧也有一个 `100%` 缩放控件
- **应该统一成一处**

### 6. 中英文混用
- 侧边栏：全中文（选择、上传、文字…）
- 底部 AI 工具：全英文（Remove BG、Upscale、Eraser…）
- 应统一语言

### 7. "已保存" Badge 颜色与主题割裂
- 使用了 `emerald-500` 绿色，但项目主色调是 indigo/violet
- 在暗色背景下虽然可读，但与整体色调不协调

## 🟢 需要增强

### 8. 按钮交互 — 缺少 active/pressed 状态反馈
- 左侧栏 "选择" 按钮有高亮 (`secondary` variant)，但视觉对比度不够
- 和未选中按钮的差别太小

### 9. AI 聊天面板 — 内容过于空旷
- 只有一条欢迎消息，然后大片空白
- 应该添加快捷操作卡片或建议标签来填充空间

### 10. 字体 — 使用 Geist 是好的，但 heading/body 没有层次
- 所有文字都是同一字体、相近大小
- TopBar 的 "WebPoster" 品牌名应该更突出

## 🛠 修复计划

| 优先级 | 问题 | 修复方案 |
|--------|------|----------|
| P0 | 左侧栏图标截断 | 给侧边栏内层容器添加正确的 `px-2` 内边距 |
| P0 | 快速素材无边距 | 修复 padding |
| P0 | 底部工具栏无区隔 | 添加明确的背景色 + border-top |
| P1 | 重复缩放显示 | 移除画布浮动 badge，保留底部控件 |
| P1 | 右侧面板 border 不可见 | 增强 border 可见度 |
| P1 | 中英文混用 | 统一为中文 |
| P2 | 已保存 badge 颜色 | 改为与 primary 色系一致 |
| P2 | 聊天面板空旷 | 添加建议操作卡片 |
